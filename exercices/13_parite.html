Examinez le programme parite.js ci-dessous qui afficheles nombrees pairs (divisibles par 2) jusqu'à 10.

  • for (var i=1; i<= 10; i++){
  • if (i % 2 === 0) {
  • console.log(i + "est pair");
  • }
  • }

il utilise l'opérateur module %, qui calcule le reste de la division enteère d'un nombre par un autre .si le résultat du modulo d'un nombre par 2 est 0, alors ce nombre est pair.Voici quelques exemple d'utilisation de ce t opérateur.

  • console.log(10 % 2); // Affiche 0 car 10=5*2 + 0
  • console.log(11 % 2); // Affiche 1 car 10=5*2 + 1
  • console.log(18 % 3); // Affiche 0 car 18=6*3 + 0
  • console.log(20 % 3); // Affiche 2 car 20=6*3 + 2
